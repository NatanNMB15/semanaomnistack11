# Mobile Be The Hero

## 1. Instalação

Necessário estar com o Node JS versão 12 instalado e o Expo CLI.

Executar o comando abaixo para instalar as dependências:

Utilizando Yarn:

```
yarn install
```

Utilizando NPM:

```
npm install
```

## 2. Executar no ambiente de desenvolvimento

Inicie o Expo com o seguinte comando:

Utilizando Yarn:

```
yarn start
```

Utilizando NPM:

```
npm start
```
